What can I edit and what should I never touch?

In the Admin section you'll see a lot of objects, like:

  • Groups
  • Users
  • Registrants
  • Etc...
Please never touch Groups or Site.
You can edit Users but don't create any Users, they are automatically created when a new Registrant gets made. Basically every real human being gets a User, which can hold several Registrants. For example, if Michela Dai Zovi aka Dahmernatrix goes to RollerCon every year, every time you add a new Registrant (ie MVP 2015, Skater 2016, etc) with the same email address as one before (mdaizovi@gmail.com), it gets added to the User assosciated with that email address. It would be possible to manually connect registrants with different email addresses, like if someone changes their email address, but lets pretend it's impossible because it's messy.

 

How do I input new Registrants (Attendees)?

You can input new attendees thorugh the Admin. On the top right of the screen, under the blue bar you should see two little grey buttons: Import, and Add registrant.
Use Add registrant if you just want to manually add a person or two. Please note that You have to input email, first_name, and last_name. Sk8name and Sk8number are optional, but if they do have a sk8number please enter it because that becomes their User display name forever. If they don't have one, like if they are a vendor, their first name will be the display name.
If you need to add a lot of people, use Import. You don't need to massage the data-- as long as it always looks like it did in 2016 it will translate how BPT says it how I want it-- but you need to make the excel sheet present the date in the order the computer expects it to be in.
To get an example of what it wants, export some existing registrants, because the order/format is the same for import and export.
Depending on what we decide re: ease of accidentally deleting everyone :-) the import button will either be by the import/add registrant buttons, or on the left from an Action dropdown, in the same place you can delete registrants.
Either way, export the registrant and it'll show you the order it wants. I tried to make it mirror the BPT reports as closely as possible.
To format the BPT Excel sheet:

  1. Take your BPT report, delete the first few rows down to and including "Will Call Tickets" so row 1 is the columns names, row 2 starts the data.
  2. Insert a column so that Column A is blank and Column B is the BPT Ticket ID.
  3. Delete Columns C-M (order date/event start date/event end date/attendee l name/attendee f name/etc) but keep Shipping State.
  4. Keep Shipping State and Shipping County, I use those for INTL. Delete Shipping Zip, Phone, Email, Card Number, Barcode, Box Office User, Order Notes.
  5. Keep Admission Level. Delete Price, Admitted, Question 1, Question 6, Question 16.
  6. So now your order should be:blank column, BPT Ticket ID, State, Country, Admission Level, Last Name, First Name, email, skate name, skate number, skill, gender, league affiliation, insurance carrier, insurance number, age group, willing to volunteer, favorite part.
  7. Last step: columns S,T,U, and V. Leave S blank, but then for T,U, and V use the example of the exported Registrants. T is the primary key of the Con this person is registering for, probably 2 for 2016 or 3 for 2017. Just copy and paste the values for con, con__city and con__year (example, "Las Vegas", "2016").
  8. Copy and paste the header (row 1) from the exported Registrant file, so that it's my header and not the BPT header on the excel sheet.
  9. Save file as an XLS somewhere. (XLSX doesn't work right, I don't know why it's an option)
  10. Go back to Admin on site, click Import, choose XLS (XLSX doesn't work right), and import. It'll show you what it's about to import before importing.
  11. Either be stoked you just imported 1,000 records in a few minutes, or, if there's a problem, take a screenshot and call me.
